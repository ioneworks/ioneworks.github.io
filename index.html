Getting Started

  • JDK 21 준비, 다운로드 및 압축 해제 : Go JDK Download
  • IONEWORKS.zip 다운로드 및 압축 해제 : Go IONEWORKS Download
  • H2DB 파일모드(/사용자홈경로/ioneworks) 기본 설정
  • DBMS 변경 시 /configuration/application.properties에서 DATABASE.conn.root의 DB(Oracle, SqlServer, MySQL, MariaDB, PostgreSQL, H2) 연결정보 수정
  • Embedded Redis에서 외부 Redis로 변경 시 /configuration/application.properties에서 REDIS-SERVER.startup=false로 변경 후, REDIS.host 수정
  • JAVA 경로 설정 및 실행 (JAVA_HOME 경로 변경)
    • Window : statup.bat
    • Linux : startup.sh
  • 브라우져에서 http://localhost:8080 접속 (포트 변경은 startup.bat/sh 파일에서 IONEWORKS_PORT 수정)
  • root 계정으로 접속
    • 초기 아이디 : ioneworks
    • 초기 패스워드 : ioneworks
  • Preparing JDK 21, download and extraction : Go JDK Download
  • IONEWORKS.zip download and extraction : Go IONEWORKS Download
  • H2DB File Mode Default Settings
  • When changing the DBMS, modify the database(Oracle, SqlServer, MySQL, MariaDB, PostgreSQL, H2) connection information of DATABASE.conn.root in /configuration/application.properties.
  • When switching from Embedded Redis to an external Redis, change REDIS-SERVER.startup=false in /configuration/application.properties, and then modify REDIS.host.
  • Set JAVA Path and Execute (Change JAVA_HOME Path)
    • Windows : startup.bat
    • Linux : startup.sh
  • Access http://localhost:8080 in your browser (To change the port, modify IONEWORKS_PORT in the startup.bat/sh file).
  • Login with the root account:
    • Initial ID : ioneworks
    • Initial Password : ioneworks
